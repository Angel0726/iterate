# Python 面向对象


数据封装、继承和多态是面向对象的三大特点。



以下划线开头的标识符是有特殊意义的。以单下划线开头（_foo）的代表不能直接访问的类属性，需通过类提供的接口进行访问，不能用"from xxx import *"而导入；

以双下划线开头的（__foo）代表类的私有成员；以双下划线开头和结尾的（__foo__）代表 Python 里特殊方法专用的标识，如__init__（）代表类的构造函数。
